Presigner
=========

CLI allows you to create presigned URLs for S3.

## Installation

    $ bundle install
    $ rake build
    $ rake install

## Usage

    $ presigner url ACCESS_KEY_ID SECRET_ACCESS_KEY DURATION BUCKET KEY
